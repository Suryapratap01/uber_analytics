
# uber analytics

Constructed an end-to-end data pipeline on GCP processing 1M+ Uber trip records, reducing manual data
handling by 100%.
• Ingested raw NYC TLC trip data into Google Cloud Storage and orchestrated multi-step transformations using
Mage, a Python-based data pipeline automation tool.
• Architected a star schema data warehouse in BigQuery with optimized fact and dimension tables, achieving
sub-5-second query response through database performance indexing.
• Developed a Looker Studio dashboard surfacing KPIs: average fare by hour, trip volume by pickup zone, and
revenue by payment type.
